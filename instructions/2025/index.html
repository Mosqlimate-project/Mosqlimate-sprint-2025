Instructions - 2025

This section provides detailed instructions on how to participate in the sprint, access the data, and use the available tutorials.

How to Participate in the Sprint

To get started, follow these steps:

  1. Review the Sprint Rules: Carefully read the sprint rules here.
  2. Register for the Sprint: Complete your registration here.
  3. Access Data Documentation: Explore the documentation of the data provided for the challenge here.

Registering Your Model and Submitting Predictions

To register your model and submit predictions using the Mosqlimate API, refer to the Sprint Template GitHub repository. The repository includes examples of how to send your predictions using Python or R.

For more guidance, check out the following resources:

  • API Usage Documentation: Mosqlimate API Documentation
  • Mosqlient Package Documentation: Learn how to use the mosqlient package to send predictions in the tutorials here.
